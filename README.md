🎮 Tic-Tac-Toe Game

A simple and interactive Tic-Tac-Toe game built using Java (Swing) and Web Technologies (HTML, CSS, JavaScript). This project demonstrates both desktop and browser-based implementations of the classic game.

Features
Two-player gameplay (X vs O)
Win and draw detection
Real-time game status updates
Restart game functionality
Clean and responsive UI
Available in both Desktop and Web versions

Technologies Used
Java (Swing, AWT) – Desktop application
HTML5 – Structure
CSS3 – Styling and layout
JavaScript – Game logic and interactivity

Project Structure
TicTacToe/
│
├── TicTacToeGame.java    
├── index.html            
├── style.css             
└── script.js            

How to Run

Java Version
javac TicTacToeGame.java
java TicTacToeGame

Web Version
Open index.html in any browser
